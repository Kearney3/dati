# 示例文件说明

本目录包含了使用智能答题系统的示例文件和说明。

## 📄 Excel文件格式示例

### 标准格式

您的Excel文件应包含以下列（列名可以灵活调整）：

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 题目/题干 | ✅ | 问题的完整描述 | "以下哪项是正确的？" |
| 选项A | ✅ | 第一个选择项 | "选项内容A" |
| 选项B | ✅ | 第二个选择项 | "选项内容B" |
| 选项C | ⭕ | 第三个选择项（可选） | "选项内容C" |
| 选项D | ⭕ | 第四个选择项（可选） | "选项内容D" |
| 选项E | ⭕ | 第五个选择项（可选） | "选项内容E" |
| 选项F | ⭕ | 第六个选择项（可选） | "选项内容F" |
| 正确答案 | ✅ | 正确答案标识 | "A" 或 "AC" 或 "对" |
| 解析 | ⭕ | 答案解释说明 | "因为..." |
| 题型 | ⭕ | 题目类型 | "单选题"/"多选题"/"判断题"/"填空题" |

### 示例数据

#### 单选题示例
```
题目: JavaScript中哪个方法用于向数组末尾添加元素？
选项A: push()
选项B: pop()
选项C: shift()
选项D: unshift()
正确答案: A
解析: push()方法用于向数组末尾添加一个或多个元素
题型: 单选题
```

#### 多选题示例
```
题目: 以下哪些是前端框架？
选项A: React
选项B: Vue
选项C: MySQL
选项D: Angular
正确答案: ABD
解析: React、Vue、Angular都是前端框架，MySQL是数据库
题型: 多选题
```

#### 判断题示例
```
题目: HTML是一种编程语言
选项A: 对
选项B: 错
正确答案: B
解析: HTML是标记语言，不是编程语言
题型: 判断题
```

#### 填空题示例
```
题目: CSS的全称是___Style___。
正确答案: Cascading|||Sheets
解析: CSS全称是Cascading Style Sheets（层叠样式表）
题型: 填空题
```

## 🎯 支持的题型

### 1. 单选题
- **答案格式**: A, B, C, D, E, F
- **特点**: 只能选择一个正确答案
- **适用场景**: 基础知识测试

### 2. 多选题  
- **答案格式**: AB, ACD, BCE 等组合
- **特点**: 可以有多个正确答案
- **适用场景**: 综合知识测试

### 3. 判断题
- **答案格式**: 
  - 字母格式: A(对) / B(错)
  - 文字格式: 对/错, 正确/错误, Yes/No, True/False
  - 符号格式: √/×, ✓/✗
- **特点**: 只有两个选项
- **适用场景**: 概念理解测试

### 4. 填空题
- **答案格式**: 使用 `|||` 分隔多个空格答案
- **特点**: 需要输入文字答案
- **适用场景**: 记忆和理解测试

## 🔧 表头映射

系统支持智能表头识别，以下表头名称都会被自动识别：

**题目列**: 题目, 题干, 问题, 题, Question, Title
**选项列**: 选项A, A, OptionA, 选择A, ChoiceA
**答案列**: 答案, 正确答案, Answer, CorrectAnswer
**解析列**: 解析, 说明, 解释, Explanation, Analysis
**题型列**: 题型, 类型, Type, QuestionType

## 📊 批量导入技巧

### 1. 数据准备
- 确保每列数据格式一致
- 删除空行和无关数据
- 检查特殊字符和编码

### 2. 工作表组织
- 不同题型可以放在不同工作表中
- 同一工作表中题型要保持一致
- 工作表名称要清晰明了

### 3. 质量检查
- 确保答案格式正确
- 检查选项内容完整
- 验证题目描述清晰

## ⚠️ 常见问题

### 问题1: 文件上传失败
**解决方案**: 
- 检查文件格式（支持.xlsx, .xls）
- 确认文件大小不超过限制
- 验证文件未损坏

### 问题2: 表头映射错误
**解决方案**:
- 使用标准的列名
- 检查列名拼写
- 手动调整映射关系

### 问题3: 答案格式不识别
**解决方案**:
- 单选题使用单个字母(A,B,C,D)
- 多选题使用连续字母(AB,ACD)
- 判断题使用对应选项或文字

### 问题4: 中文乱码
**解决方案**:
- 使用UTF-8编码保存Excel文件
- 避免使用特殊字符
- 检查系统语言设置

## 🎨 自定义配置

### 判断题选项自定义
可以在答题设置中自定义判断题的选项文字：
- 默认: 对/错
- 可选: 正确/错误, 是/否, True/False 等

### 答题模式选择
- **练习模式**: 实时反馈，适合学习
- **考试模式**: 模拟考试，有时间限制
- **背诵模式**: 先显示答案，适合复习
- **随机模式**: 打乱题目和选项顺序

## 📈 数据统计

系统会自动统计以下数据：
- 总体正确率
- 各题型正确率  
- 答题用时分析
- 错题分布情况
- 学习进度跟踪

## 💡 最佳实践

1. **题目设计**
   - 问题表述清晰明确
   - 选项长度尽量一致
   - 避免歧义和陷阱

2. **选项设置**
   - 正确答案分布均匀
   - 干扰项设计合理
   - 避免明显错误选项

3. **解析编写**
   - 解释为什么正确
   - 说明错误原因
   - 提供相关知识点

4. **数据管理**
   - 定期备份题库
   - 版本控制管理
   - 分类整理存储
